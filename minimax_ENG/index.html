The minimax algorithm

The MAX player (▲ green) picks the move with the highest value. The MIN player (▼ orange) picks the move with the lowest value.

Alpha-beta pruning

α = the best value MAX can guarantee so far.
β = the best value MIN can guarantee so far.

If α ≥ β → prune the remaining branches

The result is always the same – but fewer nodes are examined.
