Legend

  • Start – the starting position of the search. Click and drag to move it.
  • Goal – created by right-clicking once.
  • Normal path (cost 1) – created by left-clicking on a wall.
  • Wall – created by left-clicking on an open cell.
  • Cost 3 – created by right-clicking twice.
  • Cost 5 – created by right-clicking three times.
  • Explored – cells the algorithm has already examined.
  • Frontier – cells queued to be explored next.
  • Found path – the completed path from start to goal.

Left-click toggles between wall and open cell.
Right-click cycles: goal → cost 3 → cost 5 → wall.

Tip: During A* search, hover over a cell to see its f(n) value (cost function).
